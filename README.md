# Open-blog Web Application Project

**CRUD** 機能を提供するウェブアプリケーション  
**URL:** [https://shin-openblog.com](https://shin-openblog.com)

## 概要
ユーザー認証、投稿管理、コメント、いいね機能を持つフルスタックの掲示板アプリケーションです。

---

## 技術スタック

### 🔹 Backend
- **Spring Boot 3.5.5**
- **Java 17**
- **MySQL**
- **Spring Data JPA**
- **Spring Security**
- **JWT**
- **Google OAuth 2.0**

### 🔹 Frontend
- **React 18.2.0**
- **TypeScript**
- **React Router**
- **Zustand**
- **Axios**
- **React Quill**

---

## 主な機能
- 会員登録・ログイン（**Google OAuth**対応）  
- 投稿の作成・編集・削除  
- いいね・コメント機能  
- 検索機能（人気検索語・関連検索語）  
- ユーザープロフィール管理
